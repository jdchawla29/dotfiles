# Dotfiles

Personal dotfiles for cloud development environments.

## Quick Setup

```bash
curl -sSL https://raw.githubusercontent.com/jdchawla29/dotfiles/main/install.sh | bash
```

## Manual Setup

```bash
git clone https://github.com/jdchawla29/dotfiles.git ~/dotfiles
```

```bash
cd ~/dotfiles
./install.sh
```
